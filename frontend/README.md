# DevPulse Frontend

React + TypeScript + Vite frontend for the DevPulse developer analytics system.

## Quick Start

```bash
npm install
npm run dev     # http://localhost:3100
```

## Scripts

- `npm run dev` — dev server on port 3100
- `npm run build` — type check + production build
- `npm run preview` — preview production build
- `npm run lint` — ESLint check
- `npm run lint:fix` — ESLint auto-fix
- `npm run format` — Prettier formatting

## Environment

Copy `.env.example` to `.env` and adjust values as needed. `VITE_SKIP_AUTH=true` bypasses authentication for local development.
